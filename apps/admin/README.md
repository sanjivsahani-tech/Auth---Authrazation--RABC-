# Admin App

React + Vite admin frontend.

## Install
`npm install`

## Env
Copy `.env.example` to `.env`.

## First Login Flow
1. Open `/login`
2. If first admin is not created, go to `/signup`
3. Complete admin signup
4. Continue to dashboard and then use normal login in future

## User Role Change
- Go to `Users` page and use `Edit Role` action on each row
- Role assignment is single-role only
- SuperAdmin user's role cannot be changed

## Commands
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

## URL
`http://localhost:5173`
