# User App

React + Vite user frontend.

## Install
`npm install`

## Env
Copy `.env.example` to `.env`.

## Commands
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

## URL
`http://localhost:5174`

## Module Coverage
- Categories: list/create/edit/delete
- Shelves: list/create/edit/delete
- Customers: list/create/edit/delete
- Products: list/create/edit/delete

## Permissions
- View list: `<module>:view`
- Create: `<module>:create`
- Update: `<module>:update`
- Delete: `<module>:delete`
